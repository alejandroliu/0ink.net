Local Perl packages

Published: Wed 15 May 2013
By Alejandro Liu

In Blog.

Determine what is the local PERL5LIB configuration:

LIB=$(
  for d in `tr : ' ' <<<$PERL5LIB
  do
    if [ -w $d ] ; then
      echo $d
    break
  fi
  done)
PREFIX=`dirname $LIB`

Install sequence

#PREFIX=$HOME/cpan
#LIB=$HOME/cpan/lib
tar zxvf $perl-mod-tar
cd $unpacked-src-dir
perl Makefile.PL PREFIX=$PREFIX LIB=$LIB "$@"
make
make test
make install
