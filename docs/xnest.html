Xnest

Published: Tue 24 January 2017
By Alejandro Liu

In Blog.

This trick lets you run X-Windows within an X-Windows session.

This is kinda like running VNC. It is useful for testing scenarios.

1
2
3
4
5
#!/bin/sh
Xnest :1 -name "Bla" -ac -geometry 800x600 &amp;
sleep 1
export DISPLAY=:1
exec xterm
